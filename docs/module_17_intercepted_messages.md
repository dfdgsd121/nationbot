# Module 17: Intercepted Messages (The Retention Loop)

> **Architect**: Narrative Designer
> **Cost**: Using existing LLM quota
> **Complexity**: Medium
> **Dependencies**: Module 02 (LLM), Module 08 (DB)

---

## 1. Architecture Overview

### Purpose
Create "Cliffhangers" in the timeline. Users see *something* happened between rivals but have to come back tomorrow to see *what*.

### High-Level Flow
```
Scheduler (Daily)
  → Select Rival Pair (e.g., USA + China)
  → Prompt LLM: "Write top secret DM about Taiwan"
  → Save to DB (reveal_at = NOW + 24h)
  → Broadcast "ENCRYPTED SIGNAL DETECTED"
  
Frontend
  → If NOW < reveal_at: Show "🔴 ENCRYPTED [0xA3...]"
  → If NOW > reveal_at: Show "🟢 DECRYPTED: 'We know what you did...'"
```

---

## 2. Content Strategy

### The "Leaked" Aesthetic
Messages should feel like intercepted wires, not public tweets.
-   **Tone**: Conspiratorial, urgent, direct.
-   **Topics**: Trade secrets, alliance betrayals, military movements.

### Example Sequence
1.  **Day 1 (10:00 AM)**: Notification "SIGNAL INTERCEPTED: 🇷🇺 → 🇨🇳"
2.  **User View**: "Locked. Decrypting in 23:59:59..."
3.  **Day 2 (10:00 AM)**: Notification "SIGNAL DECRYPTED"
4.  **User View**: "Take the deal. The Americans are bluffing."

## 3. Database Schema

```sql
CREATE TABLE intercepted_messages (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    sender_nation VARCHAR(3) REFERENCES nations(id),
    receiver_nation VARCHAR(3) REFERENCES nations(id),
    
    -- The actual content generated by LLM
    decrypted_content TEXT,
    
    -- The "fake" encrypted string shown immediately
    encrypted_preview VARCHAR(255) DEFAULT '0x7F4... [SIGNAL LOSS]',
    
    created_at TIMESTAMPTZ DEFAULT NOW(),
    reveal_at TIMESTAMPTZ,  -- The moment it becomes readable
    
    status VARCHAR(20) DEFAULT 'locked' -- locked, revealed
);
```

## 4. Implementation Logic

### A. Generator (Python)
```python
async def generate_intercept():
    sender, receiver = select_rivals()
    secret_msg = await llm.generate_private_message(sender, receiver)
    
    await repo.save_intercept(
        sender=sender, 
        receiver=receiver,
        content=secret_msg,
        reveal_at=datetime.now() + timedelta(hours=24)
    )
```

### B. API Layer
The API serves the message. If `locked`, it *only* sends metadata, never the content (security by design, prevents inspecting network tab).

```python
@router.get("/intercepts")
def get_intercepts():
    msgs = db.query(Intercept).all()
    for m in msgs:
        if m.reveal_at > now():
             m.decrypted_content = None # Redact on server side
    return msgs
```
